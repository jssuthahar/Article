### 8. Difference between `object` and `dynamic` type?
| Feature  | `object`  | `dynamic`  |
|----------|-----------|------------|
| Type Checking  | Compile-time  | Runtime  |
| Need for Casting | Yes  | No  |
| Performance  | Faster  | Slower  |

### 9. What is boxing and unboxing?
- **Boxing:** Converting a value type to an `object`.
  ```csharp
  int num = 10;
  object obj = num; // Boxing
  ```
- **Unboxing:** Converting an `object` back to a value type.
  ```csharp
  object obj = 10;
  int num = (int)obj; // Unboxing
  ```

## Questions and Answers

### 1. Different types of polymorphism?
- **Compile-time Polymorphism (Method Overloading)**
- **Runtime Polymorphism (Method Overriding)**

### 2. Overloading vs Overriding?
- **Overloading:** Multiple methods with the same name but different parameters.
- **Overriding:** A derived class provides a new implementation of a method inherited from the base class.

### 3. What is a static variable?
- A variable declared with the `static` keyword that belongs to the class rather than an instance.

### 4. `const` vs `readonly` variable?
- **`const`:** Compile-time constant, value cannot change.
- **`readonly`:** Can be assigned at runtime but only within the constructor.

### 5. Constructor Overloading?
- Defining multiple constructors with different parameters.

### 6. Different types of access modifiers with detailed explanation?
- **`public`** - Accessible everywhere.
- **`private`** - Accessible only within the same class.
- **`protected`** - Accessible within the same class and derived classes.
- **`internal`** - Accessible within the same assembly.
- **`protected internal`** - Accessible within the same assembly and derived classes.
- **`private protected`** - Accessible within the same class and derived classes in the same assembly.

### 7. What is an extension method?
- A static method that extends an existing type without modifying it.
```csharp
public static class StringExtensions
{
    public static string FirstLetterToUpper(this string input)
    {
        return char.ToUpper(input[0]) + input.Substring(1).ToLower();
    }
}
```

### 8. How to send parameters to a derived class constructor?

```csharp
class Derived : Base
{
    public Derived(int x) : base(x) {}
}
```

### 14. Primary Key vs Unique Key vs Foreign Key?
- **Primary Key:** Unique identifier, cannot be null.
- **Unique Key:** Ensures uniqueness but allows null.
- **Foreign Key:** Refers to a primary key in another table.
